# My Pi-hole Lists

Custom blocklists and whitelists for Pi-hole DNS filtering.

## Lists

- `adlists.list` - Ad-blocking URL lists
- `black.list` / `blacklist.txt` - Custom blocked domains
- `whitelist.txt` - Allowed domains
- `regex.list` - Regex-based blocking rules
- `sharedhosts.txt` - Shared hosts entries

## Usage

Import these lists into your Pi-hole configuration via the web interface or CLI.
